One persistent school record connects student creation, teacher judgment, and a family invitation while keeping each view deliberately different. Implemented learning loop
1

Student chooses

The signed-in student chooses a maker path and opens the current teacher-selected scaffold. There is no score, timer, streak, or rank.

2

Student creates

The learner writes a first design. Bounded validation checks the artifact without inferring ability or asking AI to write it.

3

Critique and revision

The student critiques the first design, revises it, and completes a final math self-check. The artifact remains the learner's work.

4

Teacher review

The assigned teacher sees the maker path, first design, student critique, revision, self-check, and any record challenge.

5

Next scaffold choice

The teacher chooses the next support and scaffold level. Kanni suggests a reduction but never reduces support automatically.

6

Parent handoff

The linked parent receives one reviewed invitation and returns a bounded signal. Raw artifact text stays out.

7

Changed next activity

The next cycle inherits guided, light, or independent scaffolding. Independent mode makes no model call.

The teacher's support and scaffold choices return to the learner as real behavior, so agency can grow without removing help too early.
